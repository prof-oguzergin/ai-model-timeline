# Yapay Zeka Modelleri Zaman Çizelgesi / AI Models Timeline

Büyük dil modellerinin (LLM) sürüm tarihleri ve dönüm noktalarını gösteren zaman çizelgesi. Türkçe ve İngilizce versiyonları mevcuttur.

**Canlı sayfa:** [prof-oguzergin.github.io/ai-model-timeline](https://prof-oguzergin.github.io/ai-model-timeline/)

## Çizelge

- `ai_timeline_final_tr.png` — Türkçe (yüksek çözünürlük)
- `ai_timeline_final_tr_linkedin.png` — Türkçe (LinkedIn formatı)
- `ai_timeline_final.png` — İngilizce (yüksek çözünürlük)
- `ai_timeline_final_linkedin.png` — İngilizce (LinkedIn formatı)

## Yeniden Üretme

Türkçe çizelge:
```
python ai_timeline_final_tr.py
```

İngilizce çizelge:
```
python ai_timeline_final.py
```

Çizelgenin kendisi atıf vererek serbestçe kullanılabilir.
